String Creation

Strings in JavaScript can be created using single quotes, double quotes, or backticks (template literals).
let str1 = "Hello, World!";
let str2 = 'JavaScript is fun!';
let str3 = `Template literals are great!`;

String Methods

Strings in JavaScript have a number of built-in methods that can be used to manipulate the string.
console.log("String Methods: toUpperCase, length, includes");
console.log(str1.toUpperCase()); // Converts to uppercase
console.log(str2.length); // Gets the length of the string
console.log(str3.includes("great")); // Checks if the string contains a substring

String Interpolation

String interpolation is a feature in JavaScript that allows you to embed variables or expressions within a string.
let name = "John";
let greeting = `Hello, ${name}!`; // Using template literals for interpolation
