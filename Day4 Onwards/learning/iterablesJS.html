Iterables

1. What are Iterables?

Iterables are objects that can be iterated over.

2. Using Built-in Iterables

Iterables can be created using built-in iterables.
Some common built-in iterables include:

  • Array: An array is a collection of elements that can be iterated over.
  • String: A string is a sequence of characters that can be iterated over.
  • Map: A map is a collection of key-value pairs that can be iterated over.
  • Set: A set is a collection of unique values that can be iterated over.
  • Arguments: The arguments object is an array-like object that contains the arguments passed to a function.

Example:

        let fruits = ["Apple", "Banana", "Cherry"];
        for (let fruit of fruits) {
            console.log(fruit);
        }

3. Iterating over Iterables

Iterables can be iterated over using the for...of loop.
The for...of loop allows you to iterate over the elements of an iterable object.
You can also use the forEach() method to iterate over the elements of an iterable object.

Example:

        let fruits = ["Apple", "Banana", "Cherry"];
        fruits.forEach(fruit => {
            console.log(fruit);
        });

Iterables can also be converted to arrays using the spread operator.
The spread operator allows you to expand an iterable object into an array.

Example:

        let fruits = new Set(["Apple", "Banana", "Cherry"]);
        let fruitsArray = [...fruits];
        console.log(fruitsArray); // Output: ["Apple", "Banana", "Cherry"]

Iterables can also be converted to strings using the join() method.
The join() method allows you to join the elements of an iterable object into a string.

Example:

        let fruits = ["Apple", "Banana", "Cherry"];
        let fruitsString = fruits.join(", ");
        console.log(fruitsString); // Output: "Apple, Banana, Cherry"

Iterables can also be filtered using the filter() method.
The filter() method allows you to create a new iterable object that contains only the elements that pass a specified test.

Example:

        let fruits = ["Apple", "Banana", "Cherry"];
        let filteredFruits = fruits.filter(fruit => fruit.startsWith("A"));
        console.log(filteredFruits); // Output: ["Apple"]
Note: Iterators provide a controlled way to work with data sequences, enabling custom iteration logic for various data structures.
