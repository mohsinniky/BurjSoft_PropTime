JQuery:

jQuery is a JavaScript Library, which greatly simplifies JavaScript programming, since it is easy to learn.

$(document).ready(function(){
$("code").click(function(){
$(this).hide();
});
});

What is JQuery?

The purpose of jQuery is to make it much easier to use JavaScript on your website.

        The jQuery library contains the following features:
        ->  HTML/DOM manipulation
        ->  CSS manipulation
        ->  HTML event methods
        ->  Effects and animations
        ->  AJAX
        ->  Utilities

Using JQuery:

For using JQuery, you need to include the jQuery library in your HTML document using its CDN(Content Delivery Network)

JQuery Syntax:

$(selector).action()

        A $ sign to define/access jQuery
        A (selector) to "query (or find)" HTML elements
        A jQuery action() to be performed on the element(s)

JQuery Selectors:

$(this).hide() - hides the current element.
$("p").hide() - hides all <"p"> elements.
$(".test").hide() - hides all elements with class="test".
$("#test").hide() - hides the element with id="test".

Other Selectors:

Note: all jQuery commands are to be inside a document ready event 
        $(document).ready(function(){

            // jQuery methods go here...

        });

JQuery HTML:

GET:

For getting the content/value/HTML of an HTML element, use the following jQuery method:
let variable = $(selector).html();
let variable = $(selector).text();
let variable = $(selector).val();
let variable = $(selector).attr();

SET:

For setting the content/value/HTML of an HTML element, use the following jQuery method:
$("#btn1").click(function(){
$("#test1").text("Hello world!");
});
$("#btn2").click(function(){
$("#test2").html("Hello world!");
});
$("#btn3").click(function(){
$("#test3").val("Dolly Duck");
});

Add:

Remove:

For removing an HTML element, use the following jQuery method:
$("#btn4").click(function(){
$("#test4").remove();
});
 or for emptying an HTML element, use the following jQuery method:
$("#btn5").click(function(){
$("#test5").empty();
});

Dimensions:

For getting the width/height of an HTML element, use the following jQuery method:
let variable = $(selector).width();
let variable = $(selector).height();
For Padding/Border/Margin:
let variable = $(selector).innerWidth();
let variable = $(selector).innerHeight();
let variable = $(selector).outerWidth();
let variable = $(selector).outerHeight();
