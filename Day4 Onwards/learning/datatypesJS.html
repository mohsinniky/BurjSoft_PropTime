Primitive Data Types

  • String: Represents a sequence of characters. let color = "Yellow"; let lastName = "Johnson";
  • Number: Represents numeric values (both integers and floating-point). let length = 16; let weight = 7.5;
  • Boolean: Represents a logical entity and can have two values: true or false. let isAlive = true;
  • null: Represents the intentional absence of any object value. let x = null;
  • undefined: Indicates that a variable has not been assigned a value. let y;
  • Symbol: A unique and immutable primitive value, often used as object property keys. let id = Symbol("id");

Reference Data Types

  • Object: A collection of properties, where each property is defined as a key-value pair. let person = { firstName: "John", lastName: "Doe" };
  • Array: An ordered collection of values, which can be of any type. let fruits = ["Apple", "Banana", "Cherry"];
  • Function: A block of code designed to perform a particular task. Functions can be treated as objects in JavaScript. function greet() { console.log("Hello!"); }
