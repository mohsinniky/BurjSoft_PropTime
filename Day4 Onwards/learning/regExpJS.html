1. Regular Expressions

Regular expressions (RegExp) are patterns used to match character combinations in strings in JavaScript. They are used for searching, replacing, and validating strings.

2. Creating Regular Expressions

You can create a regular expression using either the RegExp constructor or a regular expression literal. For example:

        let regex1 = new RegExp('abc'); // Using RegExp constructor
        let regex2 = /abc/; // Using regular expression literal
3. Using Regular Expressions

You can use regular expressions with methods like test() and exec() to search for patterns in strings. For example:

        let str = "Hello, world!";  
        let regex = /world/;
        console.log(regex.test(str)); // true
        console.log(regex.exec(str)); // ["world", index: 7, input: "Hello, world!"]
4. RegExp Methods

Regular expressions have several methods, including:

  • test(): Tests if a pattern exists in a string.
  • exec(): Executes a search for a match in a string and returns an array of results.
  • toString(): Returns the string representation of the regular expression.
  • compile(): Recompiles the regular expression with a new pattern.
5. RegExp Properties

Regular expressions have several properties, including:

  • source: The text of the regular expression pattern.
  • global: A boolean indicating if the regular expression is global.
  • ignoreCase: A boolean indicating if the regular expression is case-insensitive.
  • multiline: A boolean indicating if the regular expression works across multiple lines.
6. Flags in Regular Expressions

Flags are used to modify the behavior of regular expressions. Common flags include:

  • g: Global search (find all matches).
  • i: Case-insensitive search.
  • m: Multiline search.
  • u: Unicode support.
  • s: Dotall mode (allows dot to match newline characters).

For example, you can create a case-insensitive global regular expression like this :

        let regex = /abc/gi; // Case-insensitive and global search
        console.log(regex.test("ABC")); // true

Regular expressions are powerful tools for string manipulation and validation in JavaScript. They can be used for tasks like form validation, searching for patterns, and text processing.
