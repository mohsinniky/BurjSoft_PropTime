Event Handling

Events are actions or occurrences that happen in the browser, such as clicks, key presses, or mouse movements. JavaScript can respond to these events using event handlers.

Press Button below to eexecute the following code:

onclick = "this.innerHTML = Date()" // This will change the button text to the current date and time when clicked.
