Function Declaration

A function declaration defines a named function that can be called later in the code.

        function greet() {
            console.log("Hello, World!");
        }

Function Expression

A function expression defines a function that can be assigned to a variable.

        const add = function(a, b) {
            return a + b;
        };

Arrow Functions

Arrow functions provide a more concise syntax for writing functions.

        const multiply = (x, y) => x * y;

Function Invocation, Check Console

Functions can be invoked by calling their name followed by parentheses.

Parameters are the variables listed as part of a function's definition, while arguments are the actual values passed to the function when it is called.
For example, in the function function add(a, b), a and b are parameters.
When you call add(5, 10), 5 and 10 are the arguments.
