Number Creation

Numbers in JavaScript can be created using literals or the Number constructor.
let num1 = 42; // Integer
let num2 = 3.14; // Float
let num3 = Number("123"); // From string

Number Methods

Numbers in JavaScript have a number of built-in methods that can be used to manipulate the number.
console.log("Number Methods: toFixed, toExponential, toPrecision");
console.log(num1.toFixed(2)); // Converts to fixed-point notation
console.log(num2.toExponential(2)); // Converts to exponential notation
console.log(num3.toPrecision(3)); // Converts to a specified precision

Number Operations

JavaScript supports various arithmetic operations on numbers.
console.log("Number Operations: +, -, *, /, %");
console.log(num1 + num2); // Addition
console.log(num1 - num2); // Subtraction
console.log(num1 * num2); // Multiplication
console.log(num1 / num2); // Division
console.log(num1 % 5); // Modulus

Number Conversion

Numbers in JavaScript can be converted to different data types using the following methods:
console.log("Number Conversion: toString, toLocaleString, toFixed, toExponential, toPrecision");
console.log(num1.toString()); // Converts to a string
console.log(num2.toLocaleString()); // Converts to a localized string
console.log(num3.toFixed(2)); // Converts to fixed-point notation
console.log(num2.toExponential(2)); // Converts to exponential notation
console.log(num3.toPrecision(3)); // Converts to a specified precision

Number Precision

JavaScript uses double-precision floating-point format for numbers, which can lead to precision issues in some cases.
console.log("Number Precision: 0.1 + 0.2");
console.log(0.1 + 0.2); // May not equal 0.3 due to precision issues

Number Constants

JavaScript provides several constants for common mathematical operations.
console.log("Number Constants: PI, E");
console.log(Math.PI); // PI constant
console.log(Math.E); // E constant

Number Properties

JavaScript provides several properties for working with numbers.
console.log("Number Properties: MIN_VALUE, MAX_VALUE, EPSILON");
console.log(Number.MIN_VALUE); // Smallest positive number
console.log(Number.MAX_VALUE); // Largest positive number
console.log(Number.EPSILON); // Smallest positive number greater than 0

Number Math

JavaScript provides a Math object with various mathematical functions.
console.log("Number Math: Math.sqrt, Math.pow, Math.abs");
console.log(Math.sqrt(16)); // Square root
console.log(Math.pow(2, 3)); // Power
console.log(Math.abs(-5)); // Absolute value

Number Parsing

JavaScript provides methods for parsing numbers from strings.
console.log("Number Parsing: parseInt, parseFloat");
console.log(parseInt("42")); // Parses an integer from a string
console.log(parseFloat("3.14")); // Parses a float from a string

Number Validation

JavaScript provides methods for validating numbers.
console.log("Number Validation: isNaN, isFinite");
console.log(isNaN("Hello")); // Checks if value is NaN
console.log(isFinite(42)); // Checks if value is finite

Number Formatting

JavaScript provides methods for formatting numbers.
console.log("Number Formatting: toLocaleString");
console.log(num1.toLocaleString()); // Formats number as a localized string

Number Randomization

JavaScript provides methods for generating random numbers.
console.log("Number Randomization: Math.random, Math.floor");
console.log(Math.random()); // Generates a random number between 0 and 1
console.log(Math.floor(Math.random() * 100)); // Generates a random integer between 0 and 99
