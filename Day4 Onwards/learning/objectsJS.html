Object Creation

Objects in JavaScript can be created using object literals or the Object constructor.

See Console

const car = {type:"Fiat", model:"500", color:"white"};
// another way to create an object ==> const person = new Object();

Object Properties and Methods

Objects can have properties (key-value pairs) and methods (functions associated with the object).

See Console

person.firstName = "John";
person.lastName = "Doe";
person.age = 50;
person.eyeColor = "blue";

Accessing Object Properties

Object properties can be accessed using dot notation or bracket notation.

See Console

console.log("Person Object:", person.firstName);

Object Destructuring

Object destructuring allows unpacking values from objects into distinct variables.

See Console

const { firstName, lastName } = person;
console.log("First Name:", firstName);
console.log("Last Name:", lastName);

Deleting Property

Properties can be deleted from an object using the delete operator.

See Console

delete person.age;
console.log("After Deletion:", person);

Accessing Object Methods

"This" Keyword refers to a current object.
objectName.methodName() is used to call a method

Object Constructor Making

Objects can also be created using the Object constructor furnction, which is less common but still valid.
function Person(first, last, age, eye) {
this.firstName = first;
this.lastName = last;
this.age = age;
this.eyeColor = eye;
}
const myMother = new Person("Sally", "Rally", 48, "green");
const mySister = new Person("Anna", "Rally", 18, "green");

Adding a property to a created object is easy.like using mySister.nationality = "English"; The new property will be added to mySister Not to any other Person Objects. You can NOT add a new property to an object constructor
