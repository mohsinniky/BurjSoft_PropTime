Iterators

1. What are Iterators?

Iterators are objects that allow you to iterate over a collection of values.
They provide a way to access elements of a collection one at a time, without exposing the underlying structure.

2. Using Built-in Iterators

Iterators can be created using built-in iterators.
Some common built-in iterators include:

  • Array Iterator: An iterator that allows you to iterate over the elements of an array.
  • String Iterator: An iterator that allows you to iterate over the characters of a string.
  • Map Iterator: An iterator that allows you to iterate over the key-value pairs of a map.
  • Set Iterator: An iterator that allows you to iterate over the unique values of a set.
  • Generator Functions: Functions that can be paused and resumed, allowing you to create custom iterators.

Example:

        let fruits = ["Apple", "Banana", "Cherry"];
        let iterator = fruits[Symbol.iterator]();
        
        console.log(iterator.next().value); // Output: Apple
        console.log(iterator.next().value); // Output: Banana
        console.log(iterator.next().value); // Output: Cherry
        console.log(iterator.next().done);  // Output: true

3. Using Custom Iterators

Iterators can also be used with the for...of loop.
The for...of loop automatically handles the iteration process, making it easier to work with iterators.

Example:

        let numbers = [1, 2, 3];
        for (let number of numbers) {
            console.log(number);
        }

Iterators can also be used with the spread operator.
The spread operator allows you to expand an iterable object into individual elements.

Example:

        let iterable = new Set([1, 2, 3]);
        let array = [...iterable];
        console.log(array); // Output: [1, 2, 3]

Iterators can also be used with the map() method.
The map() method allows you to apply a function to each element of an iterable object and return a new iterable.

Example:

        let numbers = [1, 2, 3];
        let doubled = numbers.map(num => num * 2);
        console.log(doubled); // Output: [2, 4, 6]

Iterators can also be used with the filter() method.
The filter() method allows you to create a new iterable object that contains only the elements that pass a specified test.

Example:

        let numbers = [1, 2, 3, 4, 5];
        let evenNumbers = numbers.filter(num => num % 2 === 0);
        console.log(evenNumbers); // Output: [2, 4]

Iterators can also be used with the reduce() method.
The reduce() method allows you to apply a function to each element of an iterable object and accumulate a single result.

Example:

        let numbers = [1, 2, 3, 4, 5];
        let sum = numbers.reduce((accumulator, current) => accumulator + current, 0);
        console.log(sum); // Output: 15

Iterators can also be used with the find() method.
The find() method allows you to search for the first element in an iterable object that satisfies a specified condition.

Example:

        let numbers = [1, 2, 3, 4, 5];
        let firstEven = numbers.find(num => num % 2 === 0);
        console.log(firstEven); // Output: 2

Iterators can also be used with the every() method.
The every() method tests whether all elements in an iterable object pass a specified test.

Example:

        let numbers = [2, 4, 6, 8];
        let allEven = numbers.every(num => num % 2 === 0);
        console.log(allEven); // Output: true

Iterators can also be used with the some() method.
The some() method tests whether at least one element in an iterable object passes a specified test.

Example:

        let numbers = [1, 2, 3, 4, 5];
        let atLeastOneEven = numbers.some(num => num % 2 === 0);
        console.log(atLeastOneEven); // Output: true

Iterators can also be used with the forEach() method.
The forEach() method executes a provided function once for each let numbers = [1, 2, 3, 4, 5]; element in an iterable object.

Example:

        let numbers = [1, 2, 3, 4, 5];
        numbers.forEach(num => {
            console.log(num);
        });

Iterators can also be used with the reduceRight() method.
The reduceRight() method applies a function against an accumulator and each element in an iterable object (from right to left) to reduce it to a single value.

Example:

        let numbers = [1, 2, 3, 4, 5];
        let product = numbers.reduceRight((accumulator, current) => accumulator * current, 1);
        console.log(product); // Output: 120

Iterators can also be used with the findIndex() method.
The findIndex() method returns the index of the first element in an iterable object that satisfies a specified condition.

Example:

        let numbers = [1, 2, 3, 4, 5];
        let indexOfFirstEven = numbers.findIndex(num => num % 2 === 0);
        console.log(indexOfFirstEven); // Output: 1
